# Requests的使用

在test的目录下我们新建一个名为“requests的使用”的python脚本，并输入以下代码：

```python
import requests

url = "https://www.baidu.com"

r = requests.get(url)
print(r.status_code)
print(r.headers)
print(r.encoding)
print(r.text)
print(r.cookies)
```
